## 🆘 COMMON ISSUES & SOLUTIONS

### "Port 8080 in use"
→ Change `server.port=8081` in `application.properties`

### "Java not found"
→ Install JDK 17 from adoptopenjdk.net

### "Maven not found"
→ Use `mvnw.cmd` wrapper instead of `mvn`

### "Login fails"
→ Check browser console (F12) for errors

### "API returns 404"
→ Ensure backend is running (check console)

### "Pages show blank"
→ Access via HTTP, not file:// protocol
