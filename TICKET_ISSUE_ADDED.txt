📋 QUICK ENDPOINT REFERENCE:
═══════════════════════════════════════════════════════════════════════════

GET    /api/tickets              → Get all tickets (role-based)
GET    /api/tickets/{id}         → Get single ticket
POST   /api/tickets              → Create new ticket
PUT    /api/tickets/{id}/assign/{engineerId} → Assign to engineer
PUT    /api/tickets/{id}/start   → Start working
PUT    /api/tickets/{id}/resolve → Resolve ticket
PUT    /api/tickets/{id}/close   → Close ticket
PUT    /api/tickets/{id}/notes   → Update resolution notes
GET    /api/tickets/stats        → Dashboard statistics
GET    /api/tickets/status/{status} → Filter by status
GET    /api/tickets/health       → Health check

🚀 HOW TO USE:
═══════════════════════════════════════════════════════════════════════════

1. Run application:
   .\mvnw.cmd spring-boot:run

2. Login to get token:
   POST http://localhost:8080/api/auth/login
   Body: { "email": "admin@college.edu", "password": "admin123" }

3. Use token in requests:
   Authorization: Bearer <your_token>

4. Call any endpoint!
   GET http://localhost:8080/api/tickets
