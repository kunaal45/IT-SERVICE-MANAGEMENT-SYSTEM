⚠️ IF YOU STILL GET ACCESS DENIED:

   If not: Run .\mvnw.cmd spring-boot:run


2. Are you trying to access the right URL?
   Should be: http://localhost:8080
   NOT: http://localhost:8080/api
   NOT: http://localhost:8080/admin


3. Did the database fix work?
   • Open MySQL Workbench
   • Run: SELECT * FROM users;
   • Should show 7 users (admin, engineers, faculty)
   • If empty: Database didn't initialize properly


4. Check browser console (F12):
   • Press F12 to open Developer Tools
   • Go to "Console" tab
   • Look for error messages
   • Share the errors with me

🔐 SECURITY NOTE:
──────────────────

The application uses JWT (JSON Web Tokens) for authentication.

When you login:
  1. System checks your email and password
  2. If correct: Creates a JWT token
  3. Token is stored in browser
  4. All future requests use this token
  5. Token expires after 24 hours
