# ⚡ QUICK FIX - 10 SECONDS

## 🎯 YOUR ISSUE

You're using **PowerShell** and got this error:
```
mvnw.cmd : The term 'mvnw.cmd' is not recognized
```

---

## ✅ THE FIX (ONE CHANGE)

**Instead of:**
```powershell
mvnw.cmd spring-boot:run
```

**Type:**
```powershell
.\mvnw.cmd spring-boot:run
```

**Just add `.\` at the beginning!**

---

## 🚀 THAT'S IT!

Press **ENTER** and your project will start! 🎉
